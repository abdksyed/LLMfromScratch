Dataset

About 300 points are sampled from two noisy rings. No straight line can put one whole ring on each side.

Linear model

The formula is sigmoid(w*x + b). Its decision boundary is where w*x + b = 0, which is always a line.

ReLU layer

ReLU creates simple bends. Several ReLUs together form a flexible piecewise-linear boundary around the ring.
