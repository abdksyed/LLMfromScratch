# India-first 40B Design Study

Compact research report for Assignment 3: a proposed data, tokenizer,
post-training and evaluation plan for a hypothetical 40B India-first model.

The site is static. Open `index.html` directly or serve this directory with any
static file server.

The hero uses NASA/GSFC's *A Changing Earth at Night (Regions)* visualization:
https://svs.gsfc.nasa.gov/31099/
